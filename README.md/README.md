# HCMS
### Halal Consultant Management System

Sistem manajemen informasi yang dirancang khusus untuk memfasilitasi operasional, dokumentasi, dan alur kerja konsultasi sertifikasi halal secara terintegrasi.

---

## 🚀 Spesifikasi Teknologi (Tech Stack)

Aplikasi ini dibangun menggunakan kombinasi teknologi Python modern untuk menghasilkan performa yang ringan dan antarmuka yang responsif:

* **Bahasa Pemrograman**: `Python 3.x`
* **Antarmuka Pengguna (GUI)**: `CustomTkinter` (Modern & Dark Mode Support)
* **Penyimpanan Data (Database)**: `SQLite` (Embeded, Tanpa Konfigurasi Server)
* **Versi Aplikasi**: `v1.0 (Stable Release)`

---

## 🛠️ Fitur Utama Aplikasi

Berdasarkan arsitektur sistem, aplikasi ini mencakup modul penting berikut:
1. **Dashboard**: Ringkasan data statistik visual dan status pengajuan aktif.
2. **Client**: Manajemen database klien/pelaku usaha yang mengajukan sertifikasi.
3. **Produk**: Pendataan jenis produk, bahan baku, dan matriks ketertelusuran halal.
4. **Invoice**: Sistem penagihan, manajemen pembayaran, dan riwayat transaksi.
5. **Surat**: Pembuatan dokumen otomatis (surat tugas, rekomendasi, dll) via template.
6. **Pengaturan**: Konfigurasi sistem, backup database, dan manajemen hak akses.

---

## 💻 Cara Menjalankan (Quick Start)

1. Pastikan dependensi GUI sudah terinstal di lingkungan Anda:
   ```bash
   pip install customtkinter
   ```
2. Jalankan file utama aplikasi:
   ```bash
   python main.py
   ```
